Lyrika Test Page

Instructions

This page helps you test the API connections for the Lyrika extension. It allows you to verify that your API keys are working correctly before implementing the full extension.

Make sure you have:

  1. Created a .env file with your API keys
  2. Run node build-config.js to generate the configuration files
  3. Added test audio samples to the assets/test-audio/ directory
